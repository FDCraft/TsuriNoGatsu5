【疑难解答】
Q：如何解除锁定60帧，部分玩家跑图卡顿？
A：锁定60帧可以访问游戏设置，将垂直同步关闭即可解决。
Q：整合包启动后崩溃怎么办？
A：本整合包需要 Java17 或 Java21 来启动，请确保你的 Java 版本是正确的。如果你额外删改了模组，请排查此问题是否来源于你新安装或删除的模组，如果你没有添加或删除任何模组，请提供崩溃日志并详细说明在崩溃前你做了什么事情。
Q：进入存档后左下角弹出的红色字 ERROR 是什么？
A：如果 ERROR 后面的字样是Tag entries cannot be empty！，则属正常现象，不会影响游玩。
Q：进入存档后崩溃或提示无效的玩家数据？
A：通常情况下，玩家昵称或游戏路径含有中文可能会导致崩溃，或者是直接关闭游戏或者关闭计算机时未关闭游戏（如拔下电源）等造成强制关闭游戏的情况，又或者是添加或删除模组，可能会导致当前正在游玩中的存档损坏，唯一的解决办法是恢复至上一次的备份。
Q：安卓、Linux、Mac平台适配？
A：
Mac&Linux：需要删除 itlt 模组以正常进入游戏。此外如果需要显示 WATERFrAMEs 模组的图片与视频，你需要额外安装 VLC。
安卓：没有测试设备，本社无法遇到的问题。如果你遇到了问题，请在 Github 提出 Issue。
